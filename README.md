# ReformaIA

Plataforma de inteligência artificial para auxiliar projetos de reforma e construção civil.

## Funcionalidades

- Geração de orçamentos automáticos com base em plantas e materiais
- Sugestões de materiais sustentáveis e econômicos
- Estimativa de prazo e mão de obra
- Integração com fornecedores locais

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/ReformaIA.git
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Execute a aplicação:
   ```bash
   python main.py
   ```

## Tecnologias

- Python 3.11+
- Claude API (Anthropic)
- FastAPI
